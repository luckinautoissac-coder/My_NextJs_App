## 🆘 常见问题

### Q: 导入后看不到数据？

```
解决步骤：
1. 刷新页面（按 F5）
2. 重新打开应用
3. 检查是否选择了正确的备份文件
4. 查看浏览器控制台是否有错误
```

### Q: 导出的文件太大？

```
这是正常的！文件大小取决于：
• 消息数量（越多越大）
• 知识库文档（文档越多越大）
• 聊天记录长度

建议：
• 如文件超过 100MB，可以考虑压缩后传输
• 使用云盘而不是邮件传输大文件
```

### Q: 忘记备份了怎么办？

```
如果还能访问公司电脑：
  ↓
远程连接到公司电脑
  ↓
导出备份文件
  ↓
通过云盘/邮件发送给自己

如果无法访问公司电脑：
  ↓
联系IT部门协助
  ↓
或等待回公司后再导出
```
