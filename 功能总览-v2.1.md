## 🎮 操作方式汇总

### 话题移动（4种方式）

**方式1：拖拽到文件夹**
```
话题 → 拖到文件夹 → 文件夹高亮 → 释放
```

**方式2：右键菜单**
```
右键话题 → 移动到文件夹 → 选择文件夹
```

**方式3：批量工具栏**
```
选中话题 → 工具栏按钮 → 点击文件夹
```

**方式4：批量拖拽**
```
选中多个 → 拖动一个 → 文件夹高亮 → 释放
```

### 话题排序（2种方式）

**方式1：文件夹内排序**
```
展开文件夹 → 拖动话题 → 拖到目标话题上 → 释放
```

**方式2：文件夹外排序**
```
拖动未归档话题 → 拖到目标话题上 → 释放
```

### 文件夹操作

**创建**
```
点击 📁+ → 输入名称 → Enter
```

**重命名**
```
右键文件夹 → 重命名 → 输入新名称 → Enter
```

**排序**
```
拖动文件夹 → 拖到目标文件夹上 → 释放
```

**删除**
```
右键文件夹 → 删除 → 话题自动移出
```

## 📖 完整使用流程

### 场景1：从零开始整理话题

```
1. 创建分类文件夹

2. 移动话题到文件夹

3. 文件夹内排序
   展开文件夹
   按优先级拖拽排序

4. 调整文件夹顺序
```
