# 🔧 浏览器缓存清除指南

如果您在使用过程中遇到以下问题：
- 发送消息后没有任何AI回复
- 充值后仍然无法使用
- 界面显示异常或功能不正常
- API设置保存后没有生效

**请按照以下方法清除浏览器缓存：**

---

## 方法一：使用应用内清除按钮（推荐⭐）

1. 打开应用
2. 在右侧边栏（话题列表）的**底部**找到 **"清除所有缓存"** 按钮
3. 点击后会弹出确认对话框，点击"确定"
4. 等待页面自动刷新（约1-2秒）
5. 重新配置您的API Key并测试

---

## 方法二：浏览器快捷键清除（适用于深度使用用户）

### Windows / Linux 用户：
1. 按下 `Ctrl + Shift + Delete`
2. 在弹出的窗口中：
   - 时间范围：选择"全部时间"或"最近1小时"
   - 勾选：✅ 缓存的图像和文件
   - 勾选：✅ Cookie 和其他网站数据
   - 勾选：✅ 本地存储数据
3. 点击"清除数据"
4. 关闭所有标签页
5. 重新打开应用网址

### Mac 用户：
1. 按下 `Command + Shift + Delete`
2. 按照上述Windows步骤操作

---

## 方法三：手动清除浏览器缓存

### Chrome / Edge：
1. 点击浏览器右上角的三个点 `⋮`
2. 选择"设置"
3. 左侧菜单选择"隐私和安全"
4. 点击"清除浏览数据"
5. 选择"高级"标签页
6. 时间范围：全部时间
7. 勾选所有选项
8. 点击"清除数据"

### Firefox：
1. 点击右上角三条横线 `☰`
2. 选择"设置"
3. 左侧选择"隐私与安全"
4. 向下滚动到"Cookie 和网站数据"
5. 点击"清除数据"
6. 勾选所有选项
7. 点击"清除"

### Safari：
1. 点击顶部菜单栏"Safari"
2. 选择"偏好设置"
3. 点击"隐私"标签
4. 点击"管理网站数据"
5. 点击"全部移除"
6. 确认删除

---

## 方法四：无痕/隐私模式测试

如果清除缓存后问题仍然存在，请尝试：

1. 打开浏览器的**无痕模式/隐私模式**：
   - Chrome/Edge: `Ctrl + Shift + N` (Windows) 或 `Command + Shift + N` (Mac)
   - Firefox: `Ctrl + Shift + P` (Windows) 或 `Command + Shift + P` (Mac)
   - Safari: `Command + Shift + N` (Mac)

2. 在无痕窗口中打开应用网址
3. 重新配置API Key
4. 测试功能是否正常

---

## ⚠️ 清除缓存后需要做的事情

清除缓存后，以下数据会被删除：
- ❌ 所有对话历史
- ❌ API Key 设置
- ❌ 智能体配置
- ❌ 自定义话题

**因此您需要：**
1. ✅ 重新输入 AIHUBMIX API Key
2. ✅ 重新选择或创建智能体
3. ✅ 重新创建话题
4. ✅ 点击"测试连接"按钮验证API是否正常

---

## 🆘 如果以上方法都无效

如果清除缓存后问题依然存在，可能是以下原因：

1. **API Key 问题**：
   - 登录 [AIHUBMIX 官网](https://aihubmix.com) 检查账户余额
   - 确认 API Key 是否过期或被禁用
   - 在控制台重新生成新的 API Key

2. **网络问题**：
   - 检查是否能访问 `https://aihubmix.com`
   - 尝试关闭 VPN 或代理
   - 检查防火墙设置

3. **浏览器兼容性**：
   - 尝试更换浏览器（推荐使用最新版 Chrome 或 Edge）
   - 确保浏览器已更新到最新版本

4. **联系支持**：
   - 在应用中使用"测试连接"功能，查看具体错误信息
   - 将错误截图发送给技术支持

---

## 📝 常见错误及解决方案

| 错误提示 | 原因 | 解决方案 |
|---------|------|---------|
| "API Key 无效或已过期" | API Key 错误或已失效 | 重新生成并输入新的 API Key |
| "账户余额不足" | AIHUBMIX 账户余额为0 | 登录官网充值 |
| "请求过于频繁" | 短时间内发送过多请求 | 等待1-2分钟后重试 |
| "网络连接失败" | 网络不稳定或被拦截 | 检查网络连接，关闭 VPN |
| "API 返回格式异常" | 服务端问题 | 稍后重试或联系技术支持 |

---

**💡 提示**：建议在清除缓存前，先尝试使用应用内的"测试连接"功能，这样可以快速定位问题根源。
