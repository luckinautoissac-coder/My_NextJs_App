# 🎉 话题管理 v2.0 - 快速上手指南

## ✨ 两大核心改进

### 1️⃣ 拖拽视觉反馈
**再也不用担心拖拽失败！**

拖动话题到文件夹时，文件夹会：
- 🔵 **背景变蓝**
- ┊┊ **显示虚线边框**
- 💬 **提示"松开移入"**
- ↗️ **轻微放大**

### 2️⃣ 多选批量移动
**一次移动多个话题！**

#### 三种方式任你选：

**方式A：批量工具栏（最快）**
1. 勾选多个话题 ☑️
2. 顶部出现蓝色工具栏
3. 点击文件夹按钮
4. 完成！✅

**方式B：批量拖拽（最直观）**
1. 勾选多个话题 ☑️
2. 拖动任意一个已选话题
3. 拖到文件夹上（会显示高亮）
4. 释放鼠标 ✅

**方式C：单个拖拽（传统方式）**
1. 拖动话题
2. 文件夹高亮显示
3. 看到"松开移入"提示
4. 释放鼠标 ✅

## 🎮 5秒快速体验

1. **刷新页面** (Ctrl+Shift+R)
2. **创建文件夹** (点击 📁+ 按钮)
3. **拖动话题** 到文件夹上
4. **观察效果** 👀
   - 文件夹变蓝了吗？
   - 看到虚线边框了吗？
   - 看到"松开移入"了吗？

## 💡 小贴士

### 选择话题
- 鼠标悬停 → 显示复选框
- 点击复选框 → 选中话题
- 选中的话题 → 蓝色背景

### 批量操作
- 选中后顶部出现工具栏
- 显示已选数量
- 一键移动到任何文件夹

### 拖拽反馈
- 拖动时文件夹高亮 = 可以放置
- 没有高亮 = 不能放置
- "松开移入"提示 = 确认操作

## ⚡ 常见问题

**Q: 复选框在哪里？**
A: 鼠标悬停在话题上就会显示

**Q: 如何取消选择？**
A: 点击工具栏的"取消选择"按钮

**Q: 拖拽时文件夹没有高亮？**
A: 刷新页面再试，或检查是否拖到文件夹上

**Q: 可以选择文件夹内的话题吗？**
A: 可以！展开文件夹后可以选择

**Q: 批量移动后选择会保留吗？**
A: 不会，移动后自动清除选择

## 🎯 推荐工作流

### 场景1：整理大量话题
1. 创建几个分类文件夹
2. 勾选同类话题
3. 批量移入对应文件夹
4. 重复直到全部整理完

### 场景2：快速单个移动
1. 直接拖动话题
2. 观察文件夹高亮
3. 松开移入

### 场景3：精确移动
1. 右键点击话题
2. 选择"移动到文件夹"
3. 从列表选择目标
