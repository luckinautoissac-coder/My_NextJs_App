## 🎮 使用说明

### 拖拽操作（新方式）
1. **鼠标移到话题上**
2. **按住鼠标左键**（任意位置都可以）
3. **拖动到目标位置**
4. **释放鼠标**

**特点：**
- ✅ 无需寻找拖拽手柄
- ✅ 整个话题区域都可拖拽
- ✅ 更符合直觉
- ✅ 操作更快捷

### 管理模式使用

#### 进入管理模式
```
点击顶部 ⚙️ 图标 → 进入管理模式
```

#### 选择话题
```
☐ 点击复选框选中
☑️ 再次点击取消选中
```

#### 批量移动
```
1. 选中多个话题
2. 点击"批量移动到..."下拉框
3. 选择目标文件夹
4. 自动完成移动
```

#### 批量删除
```
1. 选中多个话题
2. 点击"删除"按钮
3. 确认删除所有选中话题
```

#### 退出管理模式
```
再次点击 ⚙️ 图标 → 退出管理模式
```

## 🔄 操作流程

### 场景1：快速拖拽排序
```
1. 鼠标移到话题上
2. 按住不放（任意位置）
3. 拖到目标位置
4. 松开鼠标
✅ 完成排序
```

### 场景2：批量整理话题
```
1. 点击 ⚙️ 进入管理模式
2. 勾选需要整理的话题
3. 选择目标文件夹
4. 自动完成移动
5. 点击 ⚙️ 退出管理模式
```

### 场景3：批量删除话题
```
1. 点击 ⚙️ 进入管理模式
2. 勾选要删除的话题
3. 点击"删除"按钮
4. 确认删除
✅ 所有选中话题被删除
```
