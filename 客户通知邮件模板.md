- ✅ **知识库完整内容**（新增）
- ✅ **所有快捷短语**（新增）

### 🎯 操作流程

#### 第一步：在公司导出数据（约 2-3 分钟）

1. 打开系统应用
2. 点击左下角「⚙️ 设置」按钮
3. 选择「💾 数据设置」
4. 查看数据概览（确认数据量）
5. 点击「导出备份文件」按钮
6. 保存下载的 JSON 文件

**重要**：请将备份文件保存到多个位置：
- ☁️ 云盘（百度网盘、阿里云盘等）
- 📧 个人邮箱
- 📱 手机（通过微信文件传输助手）

#### 第二步：在家中恢复数据（约 3-5 分钟）

1. 在家中电脑安装系统（如未安装）
2. **首先配置 API Key**：
   - 点击「设置」→「API设置」
   - 输入您的 API Key
   - 保存设置
3. 打开「设置」→「数据设置」
4. 点击「选择备份文件恢复」
5. 选择之前导出的备份文件
6. 确认导入
7. 验证数据完整性
