# 🚀 数据迁移快速清单

## 📤 公司电脑操作（导出）

```
□ 1. 打开应用
□ 2. 点击 "⚙️ 设置" → "💾 数据设置"
□ 3. 点击 "导出备份文件"
□ 4. 保存备份文件到安全位置
□ 5. 备份文件保存到多个位置：
    □ 发送到手机
    □ 上传云盘
    □ 发送到邮箱
```

**导出文件名示例**：`chat-backup-2026-02-13.json`

---

## 📥 家里电脑操作（导入）

```
□ 1. 安装应用（如未安装）
□ 2. 设置 API Key：
    - 点击 "⚙️ 设置" → "🔧 API设置"
    - 输入 API Key 并保存
□ 3. 点击 "⚙️ 设置" → "💾 数据设置"
□ 4. 点击 "选择备份文件恢复"
□ 5. 选择备份文件
□ 6. 确认导入
□ 7. 验证数据是否正常
```

---

## ⚠️ 重要提醒

- ✅ 备份文件保存多份（手机 + 云盘 + 邮箱）
- ✅ 家里电脑需要重新输入 API Key
- ✅ 导入前建议先备份家里电脑的当前数据
- ✅ 导入会覆盖所有现有数据

---

## 📊 备份内容

备份包含：
- ✉️ 所有聊天消息
- 🤖 智能体配置

- 📚 知识库和文档

**不包含**：
